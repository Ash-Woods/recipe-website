Help!

If you're having trouble accessing your saved data try the following:

Accessing via multiple devices:

  1. Ensure you are opening the webpage using Google Chrome.
  2. Double-check that 'Sync' is switched on across both devices, noting this will require you to login with the Google Account.
  3. Check that you have the Chrome Extension 'Local Storage Sync' running and that this has saved your latest updates.
  4. Finally, on the second device, ensure that you have imported your last saved state and reloaded the page (not using the 'reload' within the Chrome Extension).
